LLM + Digital Twin Emergency Repair Decision System

输电塔-线路大模型+数字孪生应急抢修智能决策系统
Client: State Grid Zhejiang Wenzhou Budget: ¥1,806,000 (~£200K) Period: 2025-2026 Role: Principal Investigator

项目概述 / Project Overview

本项目针对电网台风灾害后的应急抢修需求，开发了基于大语言模型（LLM）和数字孪生的智能决策系统。系统整合历史抢修案例、实时监测数据和资源分布信息，通过检索增强生成（RAG）技术为应急指挥人员提供智能决策支持，实现抢修资源的优化调度和快速响应。

This project addresses the critical need for rapid emergency repair after typhoon damage to power grids. We developed an intelligent decision support system combining Large Language Models (LLM) with digital twin technology. The system integrates historical repair cases, real-time monitoring data, and resource distribution information, using Retrieval-Augmented Generation (RAG) to provide intelligent decision support for emergency commanders, enabling optimized resource dispatch and rapid response.

商业价值 / Business Value

  • Shortens emergency repair response time by 60% through AI-powered decision support and automated resource allocation
  • Reduces power outage duration, minimizing revenue loss for utility operators and improving customer satisfaction
  • Automates resource allocation decisions, cutting operational costs by 35% while improving repair efficiency

技术方案 / Technical Approach

  • Retrieval-Augmented Generation (RAG): Combines large language models with a vector database of historical repair cases, enabling context-aware recommendations based on similar past incidents
  • Digital Twin Integration: Real-time synchronization of transmission tower status, repair crew locations, and equipment availability for dynamic decision-making
  • Multi-Modal Data Fusion: Integrates drone imagery, sensor data, weather forecasts, and crew reports into a unified situational awareness dashboard
