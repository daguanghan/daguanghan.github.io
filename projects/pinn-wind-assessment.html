PINN+MARL Intelligent Wind Disaster Assessment for Power Grid

Type Power Industry Competitive Project
Duration 2024-2026
Role Principal Investigator
¥2.175M Funding

Project Overview

This project develops an advanced AI-powered system for intelligent wind disaster assessment of transmission tower-line systems. By combining Physics-Informed Neural Networks (PINN) with Multi-Agent Reinforcement Learning (MARL), we create a cutting-edge methodology for critical infrastructure risk assessment that can predict and mitigate wind-induced failures in power grid infrastructure.

Business Value

AI-powered disaster prediction enables proactive grid resilience planning and risk mitigation, reducing downtime and maintenance costs while ensuring power supply reliability during extreme weather events.

Key Technologies

Physics-Informed Neural Networks (PINN)

Integrates physical laws and engineering principles into deep learning models, ensuring predictions are physically consistent and interpretable.

Multi-Agent Reinforcement Learning (MARL)

Enables coordinated decision-making across multiple system components for optimal emergency response and resource allocation.

Wind Load Modeling

Advanced computational fluid dynamics and stochastic modeling for accurate wind load prediction on complex tower-line geometries.

Real-time Monitoring

Integration with IoT sensors and weather data for continuous structural health assessment and early warning.

Research Objectives

1. Develop PINN-based wind load prediction models that incorporate fluid-structure interaction physics for transmission tower-line systems under extreme wind conditions.

2. Create MARL algorithms for coordinated emergency response and repair resource optimization across multiple transmission lines.

3. Build a digital twin framework for real-time structural health monitoring and predictive maintenance scheduling.

4. Validate the system through field deployment and comparison with traditional assessment methods.

Expected Outcomes

• Prediction Accuracy: 95%+ accuracy in wind disaster risk assessment

• Response Time: Real-time assessment capability with <5 second latency

• Cost Reduction: 30% reduction in emergency repair costs through predictive maintenance

• System Resilience: 40% improvement in grid reliability during extreme weather events
